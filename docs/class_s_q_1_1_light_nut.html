Detailed Description

Light nut, used for scene lighting.

Member Enumeration Documentation

◆ LightType

enum class SQ::LightNut::LightType
strong

Used for the light type, there are 3 types

DIRECTIONAL, an infinitly far away light so that the rays of light are running paralell, used for things like the sun.

POINT, a light which has a position and radiates in all directions, like a lightbulb

SPOT, a light which has a position, direction and cutoff angles, like a spotlight
